# Titan Prime — preview (prototip static)

Prototip static al noului magazin **Titan Prime** (single-product, mobile-first), pentru aprobarea design-ului de către client înainte de build-ul WooCommerce.

- `index.html` — landing principal
- `*.html` — pagini legale (Termeni, GDPR, Livrare, Retur, Cookies)
- `assets/css/styles.css` — stil unic
- `assets/img/` — imagini (preluate de pe site-ul actual)

Video testimonialele sunt referențiate din URL-urile live de pe `shop.andreimarincean.ro` (temporar, pentru preview).

## Deploy
Static, fără build. Servit prin **Cloudflare Pages** (output dir = rădăcina repo-ului).

## Local
Deschide `index.html` sau folosește Live Server în VS Code.
